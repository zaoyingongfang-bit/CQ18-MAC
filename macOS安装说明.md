# macOS 版安装说明

## ⚠️ 重要：首次运行提示"无法验证开发者"的解决方法

由于应用未经过苹果公证，macOS 会阻止运行。这是正常现象，请按以下方法解决：

### 方法1：系统设置中允许（推荐）

1. 双击应用时会提示"无法验证开发者"
2. 打开"系统设置" -> "隐私与安全性"
3. 往下滚动，找到"仍要打开"按钮
4. 点击"仍要打开"，输入密码确认
5. 以后就能正常使用了

### 方法2：右键打开

1. **不要双击应用**
2. 右键点击应用图标，选择"打开"
3. 在弹出的提示中点击"打开"按钮
4. 以后就能正常使用了

### 方法3：终端命令（如果上述方法无效）

如果仍然提示"已损坏"，请使用终端命令：

1. 打开"终端"（在"应用程序 -> 实用工具"中）
2. 输入以下命令并回车：

```bash
sudo xattr -rd com.apple.quarantine /Applications/造音工坊声卡调试工具.app
```

> 如果应用在其他位置，请修改路径，或直接把应用图标拖到终端窗口自动填入路径

3. 输入 Mac 登录密码（输入时不显示，输完直接回车）
4. 完成后即可正常打开应用

## 安装步骤

### 从 DMG 安装
1. 双击下载的 `.dmg` 文件
2. 将"造音工坊声卡调试工具"图标拖到"应用程序"文件夹
3. 按上述方法解除安全限制
4. 在"应用程序"中打开

### 从 ZIP 安装
1. 解压下载的 `.zip` 文件
2. 将解压出的 `.app` 移动到"应用程序"文件夹
3. 按上述方法解除安全限制
4. 在"应用程序"中打开

## 常见问题

**Q: 为什么会提示"已损坏"？**  
A: 因为应用没有苹果开发者签名和公证。这是正常的，使用上述方法解除限制即可。

**Q: 这样做安全吗？**  
A: 是的。应用是开源的，可以查看源代码。只是绕过了 macOS 的 Gatekeeper 检查。

**Q: 点击 EXE 相关按钮为什么提示"暂未提供"？**  
A: 那些是 Windows 专用程序，macOS 版本会友好提示。其他所有功能（视频教程、图片等）都正常使用。
